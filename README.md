# 「Pokémon Battle Simulator」
**制作・立案**　佐藤 侑牙（サトウ ユウガ）
# 概要

私が通っている、バンタンゲームアカデミー高等部では年に数回**審査会**があり、二年次の一回目（2021年 9月末）の審査会に向けて作成した作品です。

コンソール向けに発売されている、「**ポケモン**」をソーシャルゲームの形で再現したゲームです。
開発期間：2021年 8月～9月

※非営利目的です※
